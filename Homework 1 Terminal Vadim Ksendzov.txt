1. pwd - посмотреть, где я
2. mkdir hw1 - создать папку с название hw1
3. cd hw1 - зайти в папку с названием hw1
4. mkdir d1 d2 d3 - создать в папке hw1 3 папки с перечисленными названиями
5. cd d1 - зайти в папку d1
6. touch 1.txt 2.txt 3.txt 1.json 2.json - создать в этой папке 5 файлов (3 txt, 2 json)
7. mkdir dd1 dd2 dd3 - создать еще 3 папки внутри папки d1
8. ls - вывести содержимое папки d1
9. vim 1.txt - открыть файл через vim
   i - установить курсор для печати (insert)
10. вводим текст
11. esq - выйти из режима ввода
    shift+: - установить курсор для команды
    wq - ввести команду для сохранения и выхода
12. cd .. - выйти из папки на уровень выше
13. mv d1/1.txt d2/1.txt - переместить файл из одной папки в другую
    mv d1/1.json d3/1.json - переместить еще один файл из одной папки в другую папку
14. cp d1/2.txt d2/2.txt - скопировать файл и вставить в другую папку
    cp d1/2.json d3/2.json - скопировать еще один файл из одной папки и вставить в другую папку
15. find d1 -type f -name "2.txt" - найти файл по имени
16. tail -f d2/1.txt - посмотреть содержимое файла в реальном времени
17. head -3 d2/1.txt - вывести первые три строки 
18. tail -4 d2/1.txt - вывести последние четыре строки
19. less d2/1.txt - посмотреть содержимое длинного файла
    q - выйти из режима просмотра файла
20. date - вывести дату и время
